# PGRC Mod Template Project
This is a template Godot 4.2 project that can serve as the starting point for making your own mod for [Parking Garage Rally Circuit](https://store.steampowered.com/app/2737300/Parking_Garage_Rally_Circuit/)

## Requirements
In order to make mods for PGRC you will need:
- Your own copy of Parking Garage Rally Circuit
- The Godot editor, specifically version 4.2 which can [be downloaded here](https://godotengine.org/download/archive/4.2.2-stable/) for your platform (Windows or Linux)
- To create a fork of this repo, where you will add your own files for your mod
- Need a way to create 3D models ([Blender](https://www.blender.org/) recommended) and 2D texture images
- Some time and patience :)
